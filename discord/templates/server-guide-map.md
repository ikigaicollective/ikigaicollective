# Server Guide — where do I post what?

This is the living map of the Ikigai Collective server. Skim it once on arrival; return when a post doesn't obviously belong somewhere.

The server is organized around the **experience of being a member**, not around admin convenience. Reading the category names top to bottom is reading the arc of making something: **arrive → signal → experiment → reflect → collaborate → connect → specialize → unwind.**

We're launching with an intentionally small channel set. Rooms are added as usage warrants.

## 🏛️ Welcome & Orientation
First impression. Read-mostly. Every invited member passes through.
- `#read-me-first` — mission + guidelines. React ✅ to unlock `@Member`.
- `#server-guide` — this map. Staff-posted.
- `#introductions` — say hi. Who you are, what you're exploring.
- `#role-picker` — domains, tools, time zone, notification opt-ins.

## 📣 Signal
Low-noise, high-signal. Mute everything else if you want; don't mute this.
- `#announcements` — staff-posted server-wide news. Opt in via `@Announcements`.
- `#community-ops` — meta. Structural suggestions, channel proposals, feedback on the server itself.

## 🧭 The Loop  *(forums)*
The core of the server. Four forums mirroring the Collective's experiment model. Start a new thread per experiment. Post **where you are in your own loop right now**.
- `#design` 💭 — "What if…?" Hypotheses, sketches, framings.
- `#implement` 🔨 — "Here's what I'm building." WIP, screenshots, demos, commits.
- `#measure` 📊 — "Here's what happened." Metrics, quotes, test results, honest assessments.
- `#iterate` 🔁 — "Here's what I'm changing." Pivots, re-hypothesizing, abandonments.

## 📓 Lab Notes
Longer-form reflection and output. The distilled version of what you're building.
- `#lab-notes` — forum. One thread per post. Use the pinned template. Tag by domain.
- `#showcase` — finished-ish output. Launches, published pieces, releases.
- `#seeking-feedback` — explicit critique requests **only**. Posting opts you in to direct response.

## 🤝 Collaboration
Finding each other and helping each other.
- `#pair-with-me` — "Looking for a co-builder / collaborator / thought partner on X."
- `#help-desk` — "I'm stuck on Y." Ask early.
- `#tool-swap` — AI tool recs, prompt patterns, stack reviews, what's working / not.

## 🎙️ Live
The human-connection counterweight to async text.
- 🔊 `Open Studio` — always-on voice room. No agenda. Drop in to co-work.
- 🎤 `Lab Night` (stage) — **weekly share-out, Thursdays 18:00 UTC**. Members present what they built / measured / iterated that week. 3–5 min each, open Q&A. Opt-in via `@Lab Night`.

## 🧠 Domains  *(forums)*
Interest-based rooms that map 1:1 with the domain roles. Launched with four — more added as members arrive.
- `#art-design`, `#code-engineering`, `#writing-publishing`, `#contemplative`.

## 💬 Lounge
The kitchen of the studio. Off-topic welcome here — keeps the working channels on-topic.
- `#water-cooler` — general chat.
