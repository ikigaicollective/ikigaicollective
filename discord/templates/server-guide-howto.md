# How to post — a short guide

## If you don't know where to post

A quick decision tree:

**1.** Is it a specific experiment or project you're working on?
→ One of **The Loop** forums (`#design`, `#implement`, `#measure`, `#iterate`), whichever stage matches.

**2.** Is it a distilled write-up of something you learned?
→ `#lab-notes`. Use the pinned template.

**3.** Looking for a collaborator, or stuck on something?
→ `#pair-with-me` or `#help-desk`.

**4.** Want structured critique on finished work?
→ `#seeking-feedback` (opts you in to direct response) or `#showcase` (share without asking).

**5.** Tool / prompt / AI stack question?
→ `#tool-swap`.

**6.** Off-topic or social?
→ `#water-cooler`.

**7.** Still not sure?
→ Default to `#lab-notes`, or tag `@Onboarding Buddy` and ask.

## Conventions

- **Threads > long messages.** Forum channels are the home for multi-day discussions. Start a thread for anything that'll outlive a single session.
- **Tags.** Use them as you post, not as an afterthought.
  - The Loop forums use **workflow tags**: `open / in-progress / blocked / shipped / archived`.
  - `#lab-notes` uses **domain tags**: art-design, code-engineering, writing-publishing, music-audio, film-video, science-research, education, entrepreneurship, wellness, contemplative, games-play, hardware-robotics.
  - Domain forums use `open / in-progress / showcase / archived`.
- **Pings.** No `@everyone` or `@here`. Use role pings sparingly. Opt into `@Announcements`, `@Lab Night`, `@Co-work`, `@Opportunities` via `#role-picker`.
- **AI authorship.** For substantially AI-generated content, mention it briefly. Clarity, not shame.
- **Feedback is consent-based.** Default to kindness and questions; save critique for `#seeking-feedback`.
- **Time zones.** Three zone roles (`@Americas`, `@EMEA`, `@APAC`) live in `#role-picker` — pick one so others know when you're reachable.

## Rituals

- **Lab Night** — Thursdays 18:00 UTC on the `Lab Night` stage. Opt in via `@Lab Night`.
- **Open Studio** — drop in any time. No agenda. Body-doubling / co-working friendly.

## The space evolves

This guide is living. If a channel is missing, redundant, or misplaced, post in `#community-ops`. The server will grow into the full tree — more Domains, Resources, GitHub mirror, Focused Sprint voice rooms — as the community fills it out.
